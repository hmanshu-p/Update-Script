# Update Script

A simple script that keeps your Linux computer up to date with a single command.

## What it does

- Optionally connects to NordVPN before updating
- Downloads and installs the latest updates for all your apps and software
- Cleans up any junk files left behind after updating
- Lets you know if your computer needs a restart

## How to use

Open a terminal and run:

```bash
~/update.sh
```

That's it. The script will walk you through the rest.

## Notes

- Works on Ubuntu, Linux Mint, Kali, and other Debian-based systems
- Saves a log of every update run to `~/.update.log` so you can see what changed
